Это стартовый шаблон для верстки!
Этот шаблон предназначен для верстки сайтов малой и средней сложности.
Он включает:
	-удобную файловую систему документов
	-базовые элементы, т.к normalize.css, шрифт fontawesome, modernize.js
	-конфиг для компилятора Prepros 6 (скачать -https://prepros.io/)
	-фреймворки jquery (https://jquery.com), slick (http://kenwheeler.github.io/slick/), ajax-отправка форм
Содержание:
fonts - шрифты шаблона
images - фотоконтент шаблона и используемых
js - используемые скрипты 
styles - основные стили шаблона
	_misc - базовые стили
		-fonts.scss - подключение шрифтов
		-normalize.scss  - сброс стилей
	plugins - плагины, расширения (fontawesome, slick, fancybox)
        sections - стили частей шаблона (header, footer, content etc.). 
        Добавляйте сюда файлы стилей других частей шаблона, если они будут (sidebar, blog, etc.)
style.scss - главный файл препроцессора стилей. Вписывайте в него основные переменные вашего макета (цвета, размеры),
а также подключайте в нем новые файлы стилей.
